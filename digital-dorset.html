Calendly Integration Option

Alongside the built-in booking system, you can embed a free Calendly widget so residents can book directly from champion calendars. Below is a live-style preview and the code to activate it.

✅ Built-in Booking System (this app)

  • Fully self-contained — no third-party account needed
  • Attendance tracker & DNA rate alerts built in
  • Project Support controls all session slots
  • All resident data stays within your control
  • Works offline / on local intranet

📅 Calendly Embed Option

  • Residents book directly into champion's live calendar
  • Automatic email & SMS confirmations sent
  • Free Calendly plan available (paid for SMS)
  • Requires internet connection to load widget
  • No built-in attendance tracking (mark manually)

How to activate the Calendly embed in this app

Step 1: Create a free account at calendly.com — use your team email address

Step 2: Create an event type: e.g. "30-min Digital Skills Drop-in" — set your available hours and location

Step 3: Go to Share → Embed → Inline Widget and copy your unique event URL

Step 4: Replace YOUR-CALENDLY-LINK in the code below and paste it into this HTML file where you want the widget to appear:

<!-- Calendly inline widget --> <div class="calendly-inline-widget" data-url="https://calendly.com/YOUR-CALENDLY-LINK" style="min-width:320px;height:630px;"></div> <script type="text/javascript" src="https://assets.calendly.com/assets/external/widget.js" async></script>

Multiple champions: each champion creates their own Calendly event type and you link their library's "Book" button to their personal URL.

💡 Recommendation

Use the built-in booking system as your primary tool — it keeps all resident data within your control and gives you the attendance tracker. Use a Calendly link as a supplementary option for residents who prefer self-booking from home (e.g. link it from text confirmations or your council webpage).
